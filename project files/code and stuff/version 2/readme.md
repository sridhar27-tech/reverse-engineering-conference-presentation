## 🐳 Docker Support

Build and run the advanced services:

```bash
# Build all stages
docker build -t neuroguard-xai:latest .

# Run MLOps Pipeline
docker run --rm neuroguard-xai:pipeline

# Run FastAPI Inference Server
docker run -p 8080:8080 neuroguard-xai:api

# Run Clinician XAI Dashboard
docker run -p 8501:8501 neuroguard-xai:dashboard

# Production inference
docker run -p 8080:8080 neuroguard-xai:production
```

Multi-stage builds optimize for different environments (dev, pipeline, api, dashboard).
